## About

Node.js tool to generate rarity scores for AsyncArt Blueprints art collections.

(Copyright) 2021 hstream@gmail.com

## How to run

```
yarn cli fetchCollection --id {COLLECTION_ID}
```

```
npm run cli fetchCollection --id {COLLECTION_ID}
```
